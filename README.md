# 📚 Snapdragon Wiki

Snapdragon Wiki adalah website sederhana berbasis HTML, CSS, dan JavaScript yang berfungsi sebagai ensiklopedia chipset Qualcomm Snapdragon.

Website ini dibuat untuk belajar struktur data, filtering, dan search engine sederhana di web statis.

---

## 🔧 Fitur

- 📱 Daftar chipset Snapdragon dari berbagai seri
- 🔍 Fitur pencarian chipset secara real-time
- 🧭 Filter berdasarkan kategori:
  - Entry Level
  - Mid Range
  - Upper Mid Range
  - Flagship
  - Special (XR, Modem, Automotive)
- 📄 Tampilan wiki-style sederhana
- ⚡ Ringan, tanpa framework

---

## 📁 Struktur Project
snapdragon-wiki/ ├── index.html   → tampilan utama website ├── style.css    → styling tampilan wiki ├── app.js       → logika filter & search └── data.js      → database chipset Snapdragon
---

## 🚀 Cara Menjalankan

1. Download atau clone repository ini
2. Buka folder project
3. Klik `index.html`
4. Website akan langsung berjalan di browser

Tidak perlu install apa pun.

---

## 🌐 Deploy ke GitHub Pages

1. Upload project ke GitHub repository
2. Masuk ke Settings
3. Pilih **Pages**
4. Set source ke `main` branch
5. Klik Save
6. Website akan live otomatis

---

## 📊 Data yang Digunakan

Website ini mencakup seri Snapdragon seperti:

- Snapdragon 200 / 400 Series (Entry Level)
- Snapdragon 600 Series (Mid Range)
- Snapdragon 700 Series (Upper Mid Range)
- Snapdragon 800 Series (Flagship)
- Snapdragon XR (VR/AR)
- Snapdragon X Series (Modem)
